This dataset contains 412 Chichewa–English sentence pairs used for evaluating semantic drift in pivot-based machine translation.

Source:
Derived from publicly available bilingual corpora commonly used in low-resource machine translation research (e.g., OPUS resources such as JW300 and Tatoeba).

Usage:
- Evaluation only
- No training performed

Domains:
- Education
- Health communication
- Formal informational text

Note:
This dataset is a filtered subset used for controlled experimentation and is not intended as a general-purpose benchmark.
